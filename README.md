# Fake Mail Detection Research

## 概要

本リポジトリは、卒業研究におけるフェイクメール・偽メール検知手法の研究に関連する公開ページです。

本研究では、フェイクメールや迷惑メールに含まれる特徴を分析し、メール本文や件名に現れる偽装表現・不自然な表現・誘導表現などを検知する手法について検討しています。

## 研究データの取り扱い

収集したメールは、研究目的のみに使用します。

具体的には、以下のような処理を行ったうえで分析します。

- 氏名、住所、電話番号、メールアドレスなどの個人情報を削除または匿名化します。
- 本文中のURLはクリックせず、必要に応じて `[URL]` のように置換します。
- 添付ファイルは開封せず、分析対象には含めません。
- 受信日時、件名、本文、差出人表示名などを研究用データとして整理します。
